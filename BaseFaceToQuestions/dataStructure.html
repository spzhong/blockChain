首先在有序的数组中，可以快速的找到特定的值，但是如果插入一条新数据，就必须找到插入的位置,然后将比新数据项大的数据项向后移动一位，来给新的数据项腾出空间，删除也是，这样移动很是费时，显而易见，如果要做很多的插入和删除的操作，就不应该用有序的数组。

另外一方面，链表中可以快速的添加和删除某个数据项，但是在链表中查找数据项可不容易，必须从头开始访问链表的每一数据项，知道知道该数据项为止，这个过程很慢

树这种结构，既能像链表那样快速的插入和删除，又能快速的查找；
实现一种特殊的树，二叉（搜索）树；

特点

1.插入：一个节点的左叶子节点的关键字值小于这个节点，右子节点的关键字值大于或等于这个节点。

2.删除：删除节点要从过年查找要删除的节点开始入手，首先找到节点，这个要删除的节点可能有以下三种情况需要考虑（
* 该节点是叶节点，没有子节点
* 该节点有一个子节点
* 该节点有两个子节点
）

性质

关于二叉树，包含的一些性质

  • 在二叉树中，第i层上至多有2i-1 个节点 （i>=1）
  • 深度为k多二叉树最多有2k-1 -1个节点 （k>=1）
  • 具有n个节点的完全二叉树的深度为log2n + 1

上面的完全二叉树对应存储结构有两种：
1.顺序存储结构
2.链式存储结构

1.首先来说，顺序存储结构

8 3 10 1 6 ^ 14 ^ ^ 4 7 13 ^

在顺序存储结构中，^ 表示的是没有节点，从顺序存储可以看出，若出现大量^ ， 则对空间是一种极大的浪费。

2.在二叉树中，每一个节点至多存在左右子树上，因此在链式存储结构中，每一节点的结构为

left_child data right_child

其中data为数据域，left_child和right_child称为指阵域，分别指向左孩子和右孩子。

二叉树的基础

创建二叉排序树

首先要了解二叉树的4种遍历的方式（前序，中序，后序，层次）

递归方式的创建

  • 1.按照层次的方式创建
    • 递归思想是，将一个数组的元素按照分层的排序方式进行依次添满二叉树
    • 结束条件：取值n小于最大数组长度
    • 执行条件：左叶去值范围2n+1，右叶子去值范围2n+2
Node* createTreeWithValues(int *values , int n , const int LEN)
{
    //给一个终止的条件
    if (n >= LEN)
    {
        return NULL;
    }
    else
    {
        Node *preNode = new Node();
        preNode->data = values[n];
        preNode->left = createTreeWithValues(values , 2*n+1, LEN);
        preNode->right = createTreeWithValues(values , 2*n+2 ,LEN);
        return preNode;
    }
}
  • 2.根据有序的数组，创建一个基于中间值的分割树左小，右边大
    • 递归思想，两两分割一个数组
    • 结束条件：开始位>结束位
    • 执行条件：中间位{(开始位+(结束位-开始位))/2}
    • 左叶子节点：开始位，结束位（中间位-1）
    • 右叶子节点：开始位（中间位+1），结束位
void createTreeWithValues(Node *&pRoot, int *values , int start , int end)
{
     if (start > end)
    {
        return;
    }
    int mid = start + (end-start)/2;
    pRoot = new Node();
    pRoot->data = values[mid];
    createTreeWithValues(pRoot->left, values, start,mid-1);
    createTreeWithValues(pRoot->right, values, mid+1, end);     
    return pRoot;
}
  • 3.根据一个无序的数组，创建一个有序的二叉树，满足的条件是（左<根,根<右）
    • 递归思想，一个有序的二叉树，使其满足（左<根,根<右）
    • 按照前序的遍历思想进行创建（根，左，右）

Node * addTreeNode(Node *root, int value)
{
    if(root == NULL)
    {
        root = new Node();
        root->data = value;
    }else{
        if (value < root->data)
        {
            root->left = addTreeNode(root->left,value);
        }
        if (value > root->data)
        {
            root->right = addTreeNode(root->right,value);
        }
    }
    return root;
}
  • 4.根据一个数组，创建一个二叉树，使得所有的元素值存放在叶子节点上
    • 思想是，数组的长度作为最后一层的叶子节点，可以计算出最大的层次n，然后累积每一层的节点树（ 2n-1 ）,得出所有的节点数，然后将空的节点数插入数组前，按照层次遍历的方式，生成一个二叉树，则最后数组元素是所要的叶子节点
    • 该算法比较low

//创建一个树
Node * createMerkleTree()
{
    int values [8] = {8,3,2,1,6,14,4,7};
    int len = sizeof(values)/sizeof(values[0]);
    int allK = allTotal(0);
    
    Node *leafs[allK];
    Node *root = NULL;
    for (int i = 0;i < allK; i++)
    {
        Node *leafNode = new Node();
        if (i < allK-len) {
            leafNode->data = NULL;
        }else{
            leafNode->data = values[i-(allK-len)];
        }
        leafs[i] = leafNode;
    }
    
    
    //2n+1,为左叶子节点，2n+2,为右叶子节点
    for(int i = 0; i < (allK)/2; i++)
    {
        if (2*i+1 < allK)
        {
            leafs[i]->left = leafs[2*i+1];
        }
        if (2*i+2 < allK)
        {
            leafs[i]->right = leafs[2*i+2];
        }
    }
    
    root = leafs[0];
    
    return root;
}
  • 5.根据一个数组（全是叶子节点），创建一个二叉树，该算法比较完美，利用队列的思路
    • 利用队列的思想进行，首相将叶子节点全部入队后，然后两次pop，得出两个叶子节点，组装成根节点，然后加入队列，重复计算，至到队列的长度为1，取得根节点即可
Node *createMerkleTree3()
{
    int values [8] = {8,3,2,1,6,14,4,7};
    int len = sizeof(values)/sizeof(values[0]);
    //叶子节点
    std::queue<Node *> nodes;
    for (int i =0 ; i < len; i++) {
        Node *root = new Node();
        root->data = values[i];
        nodes.push(root);
    }
    while (nodes.size()!=1) {
        Node *p1 = nodes.front();
        nodes.pop();
        Node *p2 = NULL;
        if (nodes.empty()) {
            //将剩余的一个子结点，再次加加入队列中
            nodes.push(p1);
            continue;
        }else{
            p2 = nodes.front();
            nodes.pop();
        }
        //创建一个父节点
        Node *root = new Node();
        root->data = p1->data * p2->data;
        root->left = p1;
        root->right = p2;
        nodes.push(root);
    }
    Node *root = nodes.front();
    long size = nodes.size();
    for (int i = 0; i < size; i++) {
        nodes.pop();
    }
    return root;
}

Node* findIndexNode(Node *p , int index)
{
    if (index == 0) return p;
    std::queue<Node *> queueList;
    queueList.push(p);
    while (!queueList.empty())
    {
        index--;
        Node *preNode = queueList.front();
        queueList.pop();
        if (index == 0)
        {
            return preNode;
        }
        if (preNode->left)
        {
            queueList.push(preNode->left);
        }
        if (preNode->right)
        {
            queueList.push(preNode->right);
        }         
    }
    return NULL;
}

二叉树中的深度

递归方式

  • 二叉树的深度定义如下：从根节点到叶子节点依次经过的接点形成树的一条路径，最长路径的长度为树的深度。

    • 如果根节点为空，深度为0
    • 如果左右节点都是空的，则深度为1
    • 递归思想，二叉树的深度=max(左子树的深度，右子树的深度)+1

}

二叉树中最大的距离（二叉树的直径）

二叉树中任意两个节点都有且仅有一条路径，这个路径的长度叫做两个节点的距离。
二叉树中所有节点之间的距离最大值就是二叉树的直径。
一共有三种情况：
1.这2个节点分别在根节点的左子树和右子树上，而且他们肯定是根节点左右子树上最远的叶子节点（他们到根节点的距离=左右子树的深度）
2.这个两个节点都在左子树上
3.这个两个节点都在右子树上

递归方式

这个效率非常的低，因为计算子树的深度和最远距离是分开递归的，存在重复递归遍历的情况。其实一次递归，就可以分别计算出深度和最远距离

maxDistanceOfTree(Node *p)
{
    if(!p) return 0;
    int distance = treeDepth(p->left) + treeDepth(p->right);
    int leftDistance = maxDistanceOfTree(p->left);
    int rightDistance = maxDistanceOfTree(p->right);
    return MAX(MAX(leftDistance, rightDistance), distance);
}

二叉树中某个节点到根的路径

寻路的问题，又是查找节点的问题
定义一个存放路径的栈（不是队列，是用一个可变的数组来实现）
1.压入根节点，再从左子树中查找，如果未找到，再从右子树中查找，如果也未找到，则弹出根节点(主要别忘记了弹出压入的根节点)，再遍历栈中上一个节点
2.如果找到了，则栈中存放的节点就是路径经过的节点

    nodeVetor->push_back(p);
    //先查找左子树
    bool find = isFindNodeOnTree2(p->left,findNode,nodeVetor);
    if (!find)
    {   //再查找右子树
        find = isFindNodeOnTree2(p->right,findNode,nodeVetor);
    }
    if (!find)
    {
        //主要别忘记了弹出压入的根节点
        //如果两边都没有查到，则弹出此根节点
        nodeVetor->pop_back();
    }
    return find;
}

二叉树中两个节点最近的公共父节点

首先需要明白，根根节点是二叉树中任意两个节点的公共节点（不一定是最近的）
因此二叉树中2个节点，最近的公共节点一定在从根节点这个路径上；
所以需要分别找到从根节到这两个节点的路径，再从这两个路径找到最近的公共节点

Node* parentOfNode(Node *p, Node *a, Node *b)
{
    if (!p || !a || !b) return NULL;
    if (a == b) return a;
    //从根节点寻找路径,分别
    std::vector<Node *> *vectorA;
    isFindNodeOnTree2(p,a,vectorA);
    std::vector<Node *> *vectorB;
    isFindNodeOnTree2(p,b,vectorB);
    if (vectorA->size()==0 || vectorB->size()==0) return NULL;
    //再两个路径中寻找公共点，注意从后面开始往前推
    for(long i = vectorA->size()-1; i >= 0; i--)
    {
        for(long j = vectorB->size()-1; j >= 0; j--)
        {
            if(vectorA->at(i) == vectorB->at(j))
            {
                return vectorA->at(i);
            }
        }
    }
    return  NULL;
}

判断二叉树是否是完全二叉树

完全二叉树的定义，若设二叉树的高度为H，除了H层外，其它的各层都达到了最大数，第h层的叶子节点是从左右到右依次排布到

完全二叉树必须满足2个条件
1.如果右子树不为空，那么左子树必定不为空
2.如果右子树为空，那么排在他后面的节点必须没有孩子节点（
要分为两种情况
1.同层次后面的节点
2.同层次前面的节点的孩子节点
）

其主要思想就是利用层次遍历的原理

非递归方式

bool isCompleteBinaryTree(Node *p)
{
    if (!p)
    {
        return false;
    }
    //判断左右子树都是空，则是完全二叉树
    if (!p->left && !p->right)
    {
        return true;
    }
    //如果左子树是空，右子树不是空，则不是完全二叉树
    if (!p->left && p->right)
    {
        return false;
    }
    //按层次遍历节点，找到满足完全二叉树的条件
    //1.如果右子树不为空，那么左子树必定不为空
    //2.如果右子树为空，那么排在他后面的节点必须没有孩子节点（
    //    要分为两种情况
    //    1.同层次后面的节点
    //    2.同层次前面的节点的孩子节点
    //）
    std::queue<Node *>queueList;
    queueList.push(p);
    bool isComplete = false;
    while(!queueList.empty())
    {
        Node *oneNode = queueList.front();
        queueList.pop();
        if(!oneNode->left && oneNode->right)
        {
            return false;
        }
        if(isComplete && (oneNode->left || oneNode->right))
        {
            return false;
        }
        //右子树为空，则已经满足完全二叉树
        if(!oneNode->right)
        {
            isComplete = true;
        }
        //压入
        if(oneNode->left)
        {
            queueList.push(oneNode->left);
        }
        if(oneNode->right)
        {
            queueList.push(oneNode->right);
        }
    }
    return isComplete;
}

判断二叉树是否是满二叉树

满二叉树的定义：初了叶子节点以外，每一个节点都有左右叶子，并且叶子节点都处于最底层的二叉树

1.获取树的深度
2.获取树的叶子节点数

满二叉树的一个特性是：叶子数 = 2深度-1，可以根据这个特性判断是否是满二叉树

bool isFullBinaryTree(Node *p)
{
    if (!p) return false;
    //二叉树的深度
    int depth = treeDepth(p);
    //二叉树的叶子节点数
    int leafNum = treeLeafs(p);
    //是否满足二叉树的特性：叶子数 = 2^(深度-1)
    if (leafNum == pow(2,(depth-1)))
        return false;
    return false;       
}

判断二叉树是否是平衡二叉树

平衡二叉树的定义为：它是一颗空树或它的左右两个子树的高度差的绝对值不超过1，并且左右两个子树都是一颗平衡二叉树。平衡二叉树又叫做AVL树

//分别获取节点的深度，然后再进行左，右两边的深度比较，直至递归下去
//该方法便于立即，但是效率低下
bool isAVLBinaryTree(Node *p)
{
    if (!p)
    {
        return true;
    }
    int left = treeDepth(p->left);
    int right = treeDepth(p->right);
    if(left-right > 1 || left-right < -1)
        return false;
    return isAVLBinaryTree(p->left) && isAVLBinaryTree(p->right);
}

第二种解法
采用后续遍历的方式，遍历二叉树上的每一个节点，在遍历到一个节点之前我们就已经遍历它的左右子树，此时，记录每一个节点为根节点的树的高度，就可以一边遍历一边判断每个节点是不是平衡的。

//分别获取节点的深度，然后再进行左，右两边的深度比较，直至递归下去
//该方法便于立即，但是效率低下

bool isAVLBinaryTree(Node *p, int *depth)
{
    if (!p)
    {
        depth = 0;
        return true;
    }
    int nLeftDepth,nRightDepth;
    bool bleft = isAVLBinaryTree(p->left,&nLeftDepth);
    bool bright = isAVLBinaryTree(p->right,&nRightDepth);
    if (bleft && bright)
    {
        int diff = nLeftDepth - nRightDepth;
        if(diff > 1 || diff < -1)
        {
            *depth = 1 + (nLeftDepth > nRightDepth ? nLeftDepth:nRightDepth);
            return true;
        }
    }
    return false;
}
