  • 思路1：交换排序找出
  • 思路2：求数组的和再减去0-99的和
  • 思路3：异或运算求出

思路1：交换排序找出

双哥数组循环依次遍历，找出相等的值（效率低下，不建议使用）

思路2：求数组的和再减去0-99的和

方法巧妙，但是如果累计计算数据较多时，容易数据溢出（不建议使用）

思路3：异或运算求出

异或的一个特点是：x ^ 0 ^ 1 ^ 2 ... ^ 99 ^ 0 ^ 1 ^ 2 ... ^ 99 = x

数a两次异或同一个数b（ a = a ^ b ^ b ）仍然为原值a

a = 10100001, b = 00000110
a = a ^ b； 　　//a=10100111
b = b ^ a； 　　//b=10100001
a = a ^ b； 　　//a=00000110 

def method3(arr) :
    var a = arr[0]
    for (i = 1; i < arr.len ; i++) :
        a = a ^ arr[i] ^ (i-1)
    return a
